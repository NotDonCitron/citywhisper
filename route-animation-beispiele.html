🗺️ Routen-Animationen

Verschiedene Animationen für deine Route – wähle den Stil, der am besten passt

1. Draw-On Effekt

Die Route wird gezeichnet wie mit einem Stift. Gut für "Route wird berechnet".

2. Laufende Striche

Bewegte Punkte/Linien zeigen Richtung an. Perfekt für Navigation.

3. Pulsierendes Leuchten

Sanft pulsierende Neon-Linie. Edler Premium-Look.

4. Bewegter Marker

Ein Punkt wandert entlang der Route. Sehr verständlich für Navigation.

5. Farbverlauf Sweep

Farbverlauf bewegt sich über die Route. Modern und dynamisch.

6. Regenbogen-Welle

Bunte, fröhliche Animation. Gut für Freizeit/Event-Routen.

7. Partikel-Spur

Ein leuchtender Punkt mit Schweif. Sci-Fi/Tech-Look.

8. Atmende Linie

Sanftes Ein- und Ausblenden. Ruhig und entspannend.

📋 Integration in Leaflet

Beispiel: Draw-On Animation

// Route mit Draw-On Effekt const routeLayer = L.geoJSON(routeGeoJSON, { style: function(feature) { return { color: '#37b5ff', weight: 6, opacity: 0.9, dashArray: '500', // Gesamtlänge der Route dashOffset: '500', lineCap: 'round' }; } }).addTo(map); // Animation starten routeLayer.eachLayer(function(layer) { let length = layer._path.getTotalLength(); layer._path.style.transition = 'stroke-dashoffset 2s ease-in-out'; layer._path.style.strokeDasharray = length; layer._path.style.strokeDashoffset = length; // Nach kurzer Zeit Animation starten setTimeout(() => { layer._path.style.strokeDashoffset = '0'; }, 100); });

Beispiel: Laufende Striche (einfach)

const routeLayer = L.geoJSON(routeGeoJSON, { style: { color: '#37b5ff', weight: 5, opacity: 0.8, dashArray: '12, 18', // Strich, Abstand lineCap: 'round' } }).addTo(map); // CSS-Animation via class hinzufügen // Oder mit JavaScript: dashOffset animieren
Tipp: Für komplexere Animationen wie den "Traveling Marker" oder "Particle Trail" kannst du eine zusätzliche SVG-Overlay-Schicht über der Leaflet-Karte erstellen oder die animateMotion API mit dem Leaflet-Pfad kombinieren.
