# CityWhisper – KI-gestützter City-Guide

Dein persönlicher Begleiter für Schönau und Mannheim. Nutzt KI zur Generierung von Tourguide-Texten und ElevenLabs für die Sprachausgabe.

## Lokale Entwicklung
1. Backend starten: `python -m uvicorn backend.main:app --reload`
2. Frontend öffnen: `http://localhost:8000`
